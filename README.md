# myPhpAdmin-local
1 -> To use this project, just download it (click on the Code button dropdown and download it as a ZIP), extract it to your directory of wish.<br>
2 -> The next step is to run apache_start.bat, a command prompt window will open indicating the local server is hosted!<br>
3 -> The next step is to just  run mysql_start.bat, a command prompt window will open indicating myPhpAdmin is up!<br>
4 ->Last but not least, go to your browser and simply type ```localhost/phpmyadmin```<br>
